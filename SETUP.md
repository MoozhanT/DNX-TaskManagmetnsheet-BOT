# راهنمای راه‌اندازی «DNX Task Manager Bot»

این پروژه از دو بخش تشکیل شده:

```
DNX-TaskManagmetnsheet-BOT/
├── backend/     ← API + بات تلگرام، با پایتون + FastAPI (پورت 8000)
├── frontend/    ← پنل مدیریت با Vite + جاوااسکریپت خالص (پورت 5173)
└── SETUP.md     ← همین فایل
```

> همه‌ی دستورها برای **PowerShell** نوشته شده‌اند.

---

## پیش‌نیازها

1. **Python 3.9 به بالا** — از [python.org](https://www.python.org/downloads/) (تیک «Add Python to PATH» را بزن)
2. **Node.js (نسخه‌ی LTS)** — از [nodejs.org](https://nodejs.org/)
3. **یک بات تلگرام** — به تلگرام برو، به [@BotFather](https://t.me/BotFather) پیام بده، دستور `/newbot` را بزن،
   یک اسم و یوزرنیم برای بات انتخاب کن. در پایان یک **توکن** به شکل `123456:ABC-DEF...` می‌گیری — آن را جایی نگه دار.

```powershell
python --version
node --version
```

---

## قدم ۱: بالا آوردن بک‌اند (FastAPI + بات تلگرام)

```powershell
cd backend
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
$env:TELEGRAM_BOT_TOKEN = "توکنی که از BotFather گرفتی"
$env:SECRET_KEY = "یک-رشته-تصادفی-دلخواه"
uvicorn main:app --reload
```

اگر توکن را ندهی، بک‌اند و پنل وب همچنان بالا می‌آیند اما بات تلگرام و یادآوری‌ها غیرفعال می‌مانند
(یک هشدار در لاگ می‌بینی).

### تست سریع بک‌اند
- `http://127.0.0.1:8000/docs` → صفحه‌ی مستندات خودکار (Swagger)

**این پنجره‌ی ترمینال را باز و روشن نگه دار.**

---

## قدم ۲: ساخت اولین حساب پنل وب

بدون این قدم، هیچ‌کس نمی‌تواند وارد پنل وب شود. فقط یک‌بار لازم است.

1. `http://127.0.0.1:8000/docs` → `POST /api/admin/setup` → «Try it out»
2. یک username/password دلخواه بگذار و «Execute» بزن.

> اگر بعداً دوباره این درخواست را بزنی، خطای ۴۰۳ می‌گیری — یعنی حساب از قبل ساخته شده.

---

## قدم ۳: بالا آوردن فرانت‌اند (پنل وب)

یک **پنجره‌ی PowerShell جدید** باز کن (پنجره‌ی بک‌اند را نبند!):

```powershell
cd frontend
npm install
npm run dev
```

`http://127.0.0.1:5173` را باز کن، از لینک «راه‌اندازی اولین حساب» یا «ورود» وارد شو.

---

## قدم ۴: وصل کردن اعضای تیم به بات

هر عضو تیم باید یک‌بار در تلگرام به بات پیام `/start` بدهد تا در پنل وب («اعضا») ظاهر شود
و بتوانی تسک بهش اختصاص بدهی.

دستورهای در دسترس هر عضو داخل چت بات:
- `/addtask عنوان | 2026-07-20 10:00` — افزودن تسک برای خودش (بخش تاریخ اختیاری است)
- `/mytasks` — لیست تسک‌های باز خودش
- `/done شناسه` — تکمیل یکی از تسک‌های خودش (شناسه از خروجی `/mytasks`)

تخصیص تسک به **دیگران** فقط از پنل وب انجام می‌شود.

| `http://127.0.0.1:5173` | پنل مدیریت (چیزی که ادمین می‌بیند) |

---

## مشکلات رایج

- **پنل وب داده‌ای نشان نمی‌دهد / خطای شبکه:** مطمئن شو پنجره‌ی بک‌اند هنوز باز و روشن است.
- **بات به دستورها جواب نمی‌دهد:** مطمئن شو `TELEGRAM_BOT_TOKEN` درست ست شده و پنجره‌ی بک‌اند در لاگ‌هایش
  خطای احراز هویت تلگرام نداده باشد.
- **رمز/نام کاربری پنل وب یادت رفته:** ساده‌ترین راه در محیط توسعه: بک‌اند را متوقف کن، فایل
  `backend/app.db` را حذف کن (دیتابیس خالی و تازه با اجرای دوباره‌ی `uvicorn` ساخته می‌شود) و دوباره از
  قدم ۲ شروع کن. **توجه: این کار تمام تسک‌ها/اعضای ثبت‌شده را هم پاک می‌کند.**
